## Iniciando o projeto

1. Instale as dependencias

   ```bash
   npm install
   ```

2. iniciar o app

   ```bash
   npx expo start --lan
   ```

3. Iniciar o Json Server, dentro da pasta server:

   ```bash
   npx json-server --watch db.json --routes routes.json --port 3333 --host 0.0.0.0
   ```
